# 🌍 Преобразование координат 🌍

## 📝 Описание

Данный репозиторий содержит проект для трансформироания координат из одной системы координат в другую. В качестве веб-фреймворков применяются **Streamlit** (интерактивный интерфейс для пользователя) и **FastAPI** (серверная часть для транформации данных). Пользователи могут загрузить Excel-файл с координатами в одной системе координат и по одному клиу перевести в другую систему координат.

**Пользователи могут:**

- 🌍 Преобразовать координаты из одной системы в другую.
- 📁 Загрузить Excel-файл со своего устройства.

## 🚀 Используемые технологии

- **Python** 🐍: основной язык программирования проекта.
- **Streamlit** 🌐: фреймворк для создания веб-интерфейса.
- **FastAPI** ⚡: фреймворк для создания API сервера.
- **NumPy** 📊 : библиотеки для работы с массивами

## 📁 Структура проекта

- **`app.py`**  
  Приложение на Streamlit, позволяющее:
  - 📤 Загрузить Excel-файл для обработки сос столбцами ('Name', 'X', 'Y', 'Z')
  - 🔄 Выбрать начальную систему координат и конечную систему для преобразования

- **`main.py`**  
  Серверная часть на базе FastAPI:
  - 📥 Принимает Excel-файл в формате `UploadFile`.
  - 🌍 Преобразует координаты из одной системы в другую.

- **`requirements.txt`**  
  📜 Список зависимостей, необходимых для работы проекта.

## 🛠️ Установка и запуск

1. **Клонируйте репозиторий** или скачайте архив:

   ```bash
   git clone https://github.com/Medrodi/tranformation_coordinates_app.git
   cd 9. app
   ```

2. **Создайте и активируйте виртуальное окружение:**

   **На Windows:**

   ```bash
   python -m venv .venv
   .venv\Scripts\activate
   ```

   **На macOS/Linux:**

   ```bash
   python3 -m venv .venv
   source .venv/bin/activate
   ```

3. **Убедитесь, что у вас установлены Python 3.x и pip. Установите зависимости:**

   ```bash
   pip install -r requirements.txt
   ```

4. **В одном терминале запустите сервер FastAPI** (при необходимости замените порт):

   ```bash
   uvicorn main:app --host 0.0.0.0 --port 8000
   ```

5. **В другом терминале запустите приложение Streamlit:**

   ```bash
   streamlit run app.py
   ```

6. **Перейдите по адресу** [http://localhost:8501](http://localhost:8501) (если не меняли порт по умолчанию) и протестируйте работу приложения:

   - Выберите начальную систему координат **«Начальная система координат»** и укажите файл, либо
   - Выберите конечную систему координат **«Конечная система координат»** и создайте свой рисунок для классификации.
   - Загрузите Excel-файл **«Выберите Excel файл»**
   - Нажмите кнопку **«Преобразовать координаты»** для их преобразования и формирования отчёта
   - Нажмите кнопку  **«Скачать готовый отчет (.docx)»** для скачивания отчёта в формате DOCX

## ⚙️ Как это работает

1. 🖌️ При загрузке Excel-файла и нажатия кнопки **«Преобразовать координаты»** отправляется POST-запросом на сервер по маршруту `/process-excel/`.
2. 🖥️ **Сервер (FastAPI)** преобразует полученные координаты по формуле из одной системы координат в другую систему координат
3. 🔮 **Итоговый отчёт** можно скачать в формате DOCX

## 🔍 Подробности реализации

- **Преобразование координат:**
  - Используется формула для преобразования координат из одной системы в другую.
  - 🌍 Сначала координаты преобразуются в ГСК-2011 а далее в любую другую необходимую
